### 2. 계층
```SQL
SELECT
    LEVEL,
    PRIOR COLNAME,
    SYS_CONNCECT_BY_PATH(COLNAME,'SEP')
  FROM T
 START WITH COL IS NULL
 CONNECT BY PRIOR SON=MOM;
```
